Huffman encoding algorithm:
Project consists of two programs
1. encoding -> takes text file as an input and generates "xyz.huff"
2. decoding -> takes "compressed.huff" as input and generates "xyz.puff"

Script files:
huff.sh -> compiles and builds huff_binary
puff.sh -> compiles and builds puff_binary

Text file : The adventures of sherlock holmes
you can find the file in --> http://www.gutenberg.org/cache/epub/1661/pg1661.txt
Original size: 581kb
Compressed size:334kb
Space savings: .42
